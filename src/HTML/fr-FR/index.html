Bienvenue dans EuroLinux 7

EuroLinux 7 a une architecture robuste qui peut évoluer pour répondre à n'importe quel rôle, n'importe où. D'un poste de bureau à un serveur, un accélérateur de particules, ou même dans le cloud, EL7 réunit les meilleurs et les plus brillantes des technologies "opensource" pour produire une plate-forme construite pour vos besoins.
